Summary: Latitude vs. X

The purpose od this project was to analyze how weather/ changes as you get closer to the equater. To acccomplish this analyisis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assempling the dataset, we used Matplotlib to plot various aspects of the weater vs. lattitude. Factors we looked at included: tmeperatere, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as a part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
